The Architecture of Large-Scale Information Systems

Fast Convergence PageRank in Hadoop:

Compute PageRank in MapReduce for a reasonably large web graph using a direct node-by-node approach. Initial input and final output are stored in Amazon Simple Storage Service(Amazon S3) files and Amazon Elastic MapReduce(Amazon EMR) is used to manage the several MapReduce passes. To improve the convergence rate, compute PageRank in MapReduce for the same web graph using a Blocked Matrix Multiplication in which the number of MapReduce passes is reduced by doing nontrivial computation in the reduce steps.
